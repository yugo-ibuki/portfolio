# Accessibility Audit

Audit the current codebase for accessibility issues. Check for:

1. **Semantic HTML** — Are semantic elements used appropriately?
2. **ARIA attributes** — Are ARIA roles, labels, and descriptions correct?
3. **Keyboard navigation** — Can all interactive elements be reached via keyboard?
4. **Color contrast** — Do text and background combinations meet WCAG AA standards?
5. **Alt text** — Do all images have meaningful alternative text?

Provide a prioritized list of issues with severity levels and remediation steps.
