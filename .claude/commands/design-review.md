# Design Review

Review the current codebase for design system compliance. Check for:

1. **Color Usage** — Are colors from the design token palette? Flag any hardcoded hex values.
2. **Typography** — Do headings and body text follow the typography scale?
3. **Spacing** — Is spacing consistent with the defined spacing scale?
4. **Responsive Design** — Are layouts mobile-first and responsive?
5. **Accessibility** — Do interactive elements meet minimum size and contrast requirements?

Provide a summary of findings with specific file locations and suggested fixes.
